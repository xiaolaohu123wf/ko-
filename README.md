## 3. 使用方法

> conda create --name xxtsk python=3.10

> conda activate xxtsk

> pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

> 学习通登录，更换element内元素更换为本机屏幕下截图

> python main.py

> 学习通学习页面全屏，自动刷课
